Orbinary Web App Project Starter
--------------------------------

A boilerplate project for new web applications.

Components
----------
* Docker Compose
* **Django 1.10.x**
	*Multiple environments (dev, stage, prod)
        *
* PostgreSQL

Prerequisites
-------------
* Docker 1.12.*
* Docker Compose 1.8.*
* Python 3.5.*

Running
-------
* Customize your .env file based on .env.example
* docker-compose up
(internet connection needed the first time)
